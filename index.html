Project_Jellycat

Responsive Web

1920px, 768px, 480px 너비의 디스플레이에서 모두 구현되는 반응형 웹입니다.
랜딩 페이지는 슬라이드 갤러리와 썸네일을 클릭하면 콘텐츠를 보여주는 팝업 갤러리 등을 구현했습니다.
form과 input 태그를 활용하여 로그인과 회원가입 페이지를 제작했으며
자바스크립트를 이용해 유효성 검사 기능을 추가했습니다.
서브 페이지인 "Product" 중 "All" 페이지에는 자동 카드 갤러리와 탭 메뉴로 구성했습니다.

기여도: 100% 개인 작업
작업 기간: 2022.09.10 - 2022.09.19
